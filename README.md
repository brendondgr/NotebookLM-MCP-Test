# AlphaFold 3 Architectural Journey

## Getting Started

This project uses `uv` for Python package management.

### Prerequisites

- [uv](https://github.com/astral-sh/uv) installed on your system.

### Running the Application

1. **Start the server:**

   ```bash
   uv run app.py
   ```

2. **View the app:**
   Open your browser and navigate to [http://localhost:7450](http://localhost:7450).

## Project Structure

- `web/`: Contains the frontend HTML and JavaScript.
- `data/`: Holds the structured research content in JSON format.
- `images/`: Stores conceptual visualizations of AlphaFold 3.
- `server.py`: FastAPI server configuration.
